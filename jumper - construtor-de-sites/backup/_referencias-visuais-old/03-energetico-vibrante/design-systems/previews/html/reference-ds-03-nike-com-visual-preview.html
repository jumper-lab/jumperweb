handoff de referencia

Este DS e do site Nike, nao da pasta.

  • Insumo de performance e produto. Cruza com briefing para definir categoria, oferta e ritmo visual final.
  • Vai cruzar com o DS do formulario do cliente para criar algo novo.
  • Nao contem fluxo final, formulario final, checkout final nem pagina final do cliente.

riscos / mitigacoes

  • Produto claro sobre branco precisa sombra/recorte.
  • Volt deve ser foco, nao texto longo.
  • Nao virar ecommerce final do cliente.
