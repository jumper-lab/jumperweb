handoff de referencia

Este DS e do site Red Bull, nao da pasta.

  • Insumo de conteudo energetico e aventura. Cruza com DS do cliente para ativar ritmo sem perder estrategia.
  • Vai cruzar com o DS do formulario do cliente para criar algo novo.
  • Nao contem fluxo final, formulario final, checkout final nem pagina final do cliente.

riscos / mitigacoes

  • Muitas cores fortes podem competir.
  • Yellow em texto pequeno deve ser focus, nao copy.
  • Nao misturar portal editorial com checkout.
