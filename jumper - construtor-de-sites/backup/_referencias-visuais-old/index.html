DSs-base por personalidade visual

Esta pagina organiza os Design Systems base usados pelo fluxo oficial do Construtor de Sites Jumper. Ao receber um briefing de cliente, o sistema cruza o DS inicial do cliente com 5 DSs-base da personalidade escolhida e gera um DS final completo para construir o site.

Fluxo oficial atual
  1. Receber briefing ou arquivo do formulario do cliente.
  2. Criar o DS detalhado do cliente.
  3. Ler os 5 DSs-base ja existentes da personalidade escolhida.
  4. Cruzar o DS inicial com os 5 DSs-base e criar o DS final.
  5. Construir o site do cliente a partir do DS final.
