# Color Variations — Noomo Lab Console System

As variações mantêm a assinatura lab console, technical proof cards, build capability matrix. Elas não autorizam transformar o DS em outro estilo.

|---|---|
| Primary | #05080E |
| Secondary | #5883FF |
| Surface | #3B3C40 |
| Text | #FFFFFF |

Preview: `#05080E` + `#5883FF` sobre `#3B3C40`.

Uso recomendado: preservar operational console e console-command-bar como âncoras visuais.
