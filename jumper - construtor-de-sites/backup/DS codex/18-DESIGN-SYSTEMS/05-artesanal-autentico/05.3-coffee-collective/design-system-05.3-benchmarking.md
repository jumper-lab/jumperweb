# Benchmarking — Coffee Collective Farmline System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | producer timeline grid cria uma experiência própria em vez de template genérico. |
| Navegação | store / cafes / producers segmented nav reduz fricção no caminho principal. |
| Hero | Nordic calm hero with producer timeline and product detail close-up prova a proposta já na primeira dobra. |
| Componentes | producer-story-card, farmline-timeline, coffee-catalogue-tile, direct-trade-metric, cafe-location-row, supply-chain-footer dão vocabulário visual exclusivo. |
| Motion | timeline line grow, catalogue fade, calm image settle reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de producer-story-card devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Producer timeline, Coffee catalogue, Direct trade explanation, Cafe locations, Education journal, Subscription path, Producer stories, Footer as supply-chain map deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Torrefação nórdica / cadeia de valor
- direct-trade coffee customers and cafe visitors

## Riscos Evitados

- avoid making producers a decorative backdrop
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | producer timeline grid | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | store / cafes / producers segmented nav | O menu precisa carregar uma função do segmento. |
| Hero | Nordic calm hero with producer timeline and product detail close-up | A mídia/diagrama precisa provar a promessa. |
| Motion | timeline line grow, catalogue fade, calm image settle | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | producer timeline becomes swipeable story sequence | Mobile preserva conversão e leitura. |
