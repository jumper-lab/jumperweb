# Benchmarking — Ceremony Cupping Journal System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | cupping journal commerce cria uma experiência própria em vez de template genérico. |
| Navegação | roast finder nav with subscription shortcut reduz fricção no caminho principal. |
| Hero | journal hero with tasting cards, bag detail and roast finder action prova a proposta já na primeira dobra. |
| Componentes | flavor-note-card, roast-finder-filter, origin-story-slab, subscription-builder, brew-method-step, tasting-ledger-footer dão vocabulário visual exclusivo. |
| Motion | paper card reveal, flavor note underline, slow product hover reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de flavor-note-card devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Roast finder, Flavor note cards, Origin story, Subscription builder, Brew guide, Seasonal release, Wholesale path, Footer as tasting ledger deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Torrefação / café especial
- specialty coffee buyers and subscribers

## Riscos Evitados

- avoid mass-market cafe sweetness
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | cupping journal commerce | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | roast finder nav with subscription shortcut | O menu precisa carregar uma função do segmento. |
| Hero | journal hero with tasting cards, bag detail and roast finder action | A mídia/diagrama precisa provar a promessa. |
| Motion | paper card reveal, flavor note underline, slow product hover | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | roast finder first, tasting cards become carousel | Mobile preserva conversão e leitura. |
