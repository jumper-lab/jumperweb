# Benchmarking — Assembly Traceable Roast System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | sourcing ledger minimal cria uma experiência própria em vez de template genérico. |
| Navegação | thin text nav with roast archive access reduz fricção no caminho principal. |
| Hero | quiet product hero with origin ledger and sustainability proof table prova a proposta já na primeira dobra. |
| Componentes | origin-ledger-row, minimal-product-card, sourcing-principle-note, roast-archive-table, wholesale-enquiry-panel, provenance-footer dão vocabulário visual exclusivo. |
| Motion | subtle line draw, table row reveal, no decorative bounce reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de origin-ledger-row devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Origin ledger, Roast archive, Sourcing principles, Product education, Subscription logic, Wholesale enquiry, Journal index, Footer as provenance record deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Café minimalista / sustentabilidade
- coffee buyers who care about sourcing

## Riscos Evitados

- avoid rustic clutter that hides traceability
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | sourcing ledger minimal | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | thin text nav with roast archive access | O menu precisa carregar uma função do segmento. |
| Hero | quiet product hero with origin ledger and sustainability proof table | A mídia/diagrama precisa provar a promessa. |
| Motion | subtle line draw, table row reveal, no decorative bounce | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | ledger rows become stacked provenance cards | Mobile preserva conversão e leitura. |
