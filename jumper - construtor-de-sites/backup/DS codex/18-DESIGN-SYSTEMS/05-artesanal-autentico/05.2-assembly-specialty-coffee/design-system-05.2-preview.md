# Assembly Traceable Roast System

Categoria: 05 — Artesanal e Autêntico
Site inspiração: Assembly Specialty Coffee — https://assemblycoffee.co.uk/

Este DS não é uma pele de cores. Ele parte do padrão traceability ledger, minimal product education, sustainability proof e transforma o site em um sistema com anatomia própria: sourcing ledger minimal.

## Filosofia Visual

- Arquitetura: sourcing ledger minimal
- Navegação: thin text nav with roast archive access
- Hero: quiet product hero with origin ledger and sustainability proof table
- Ritmo: subtle line draw, table row reveal, no decorative bounce
- Responsivo: ledger rows become stacked provenance cards
- Risco principal evitado: avoid rustic clutter that hides traceability

## Revisao Funcional Codex

Erro encontrado: minimalismo sem rastreabilidade acionavel.

Funcionalidade obrigatoria adicionada ao preview HTML: Origem.

Controles agora presentes:

- seletor de modo com aria-pressed;
- cards/componentes com estado ativo;
- formulario real com input/select;
- resultado com aria-live;
- paleta clicavel para copiar token;
- secoes do DS listadas como cobertura funcional.

Criterio de rejeicao: se o DS voltar a parecer cartaz estatico ou skin de outro DS, ele deve ser refeito antes de qualquer importacao oficial.
