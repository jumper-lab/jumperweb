# Benchmarking — Leading Learning Pathways System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | pathway map modular cria uma experiência própria em vez de template genérico. |
| Navegação | rounded topic tabs with parent resource shortcut reduz fricção no caminho principal. |
| Hero | paper-collage mission hero with pathway cards and caregiver proof prova a proposta já na primeira dobra. |
| Componentes | pathway-step-card, caregiver-note, resource-filter-chip, impact-meter, download-card, training-module-row dão vocabulário visual exclusivo. |
| Motion | gentle paper slide, staggered path drawing, no bounce reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de pathway-step-card devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Mission path, Age-group pathways, Family resource finder, Educator training modules, Impact story cards, Download library, Donation/support prompt, Footer as resource map deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Educação infantil / instituição familiar
- parents, educators and nonprofit partners

## Riscos Evitados

- avoid toy-like colors that reduce credibility
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | pathway map modular | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | rounded topic tabs with parent resource shortcut | O menu precisa carregar uma função do segmento. |
| Hero | paper-collage mission hero with pathway cards and caregiver proof | A mídia/diagrama precisa provar a promessa. |
| Motion | gentle paper slide, staggered path drawing, no bounce | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | accordion pathway map and bottom resource shortcut | Mobile preserva conversão e leitura. |
