# Benchmarking — Pretty Patty Sticker Pop System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | sticker-poster grid cria uma experiência própria em vez de template genérico. |
| Navegação | badge nav orbiting menu categories reduz fricção no caminho principal. |
| Hero | poster-like hero with badge headline, mascot/sticker field and menu CTA prova a proposta já na primeira dobra. |
| Componentes | sticker-menu-card, badge-cta, loyalty-stamp-row, photo-booth-tile, mascot-callout, party-order-form dão vocabulário visual exclusivo. |
| Motion | pop scale, sticker peel, playful rotate under 3deg reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de sticker-menu-card devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Patty legend, Menu stickers, Combo stories, Loyalty club, Photo booth grid, Event/catering prompt, Order panel, Footer as sticker sheet deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Restaurante jovem / pop diner
- young diners, groups and social-first customers

## Riscos Evitados

- avoid childish UI that hurts ordering clarity
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | sticker-poster grid | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | badge nav orbiting menu categories | O menu precisa carregar uma função do segmento. |
| Hero | poster-like hero with badge headline, mascot/sticker field and menu CTA | A mídia/diagrama precisa provar a promessa. |
| Motion | pop scale, sticker peel, playful rotate under 3deg | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | stacked stickers with bottom order badge | Mobile preserva conversão e leitura. |
