# Benchmarking — Monolith Ink Portfolio System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | portfolio monolith wall cria uma experiência própria em vez de template genérico. |
| Navegação | artist index rail with booking rules reduz fricção no caminho principal. |
| Hero | near-black immersive mark, massive type and rotating artist index prova a proposta já na primeira dobra. |
| Componentes | artist-index-rail, portfolio-wall-tile, flash-sheet-card, booking-rule-panel, aftercare-accordion, archive-stamp-footer dão vocabulário visual exclusivo. |
| Motion | slow 3D mark drift, portfolio mask reveal, magnetic artist rail reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de artist-index-rail devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Artist index, Portfolio wall, Flash availability, Booking rules, Studio ethos, Aftercare guide, Consultation form, Footer as archive stamp deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Estúdio de tatuagem / arte
- culture-forward clients choosing artists by style

## Riscos Evitados

- avoid nightclub neon that cheapens tattoo authorship
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | portfolio monolith wall | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | artist index rail with booking rules | O menu precisa carregar uma função do segmento. |
| Hero | near-black immersive mark, massive type and rotating artist index | A mídia/diagrama precisa provar a promessa. |
| Motion | slow 3D mark drift, portfolio mask reveal, magnetic artist rail | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | portfolio wall becomes vertical artist archive | Mobile preserva conversão e leitura. |
