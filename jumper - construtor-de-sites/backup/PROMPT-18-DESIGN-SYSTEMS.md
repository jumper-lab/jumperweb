# PROMPT: CRIAR 18 DESIGN SYSTEMS (3 POR CADA PERSONALIDADE)

**Objetivo:** Criar 18 Design Systems completamente distintos — 3 para cada uma das 6 personalidades visuais (Acolhedor, Premium, Energético, Técnico, Artesanal, Moderno). Cada DS é baseado em um site premiado/famoso diferente.
