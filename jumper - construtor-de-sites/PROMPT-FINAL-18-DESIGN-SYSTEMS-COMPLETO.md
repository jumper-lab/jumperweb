# 🎯 PROMPT DEFINITIVO: CRIAR 18 DESIGN SYSTEMS PREMIUM DO ZERO

> **Versão:** 1.0 Final  
> **Objetivo:** Gerar 18 Design Systems completamente distintos — 3 para cada uma das 6 personalidades visuais (Acolhedor, Premium, Energético, Técnico, Artesanal, Moderno). Cada DS é baseado em um site premiado/famoso diferente do mundo real, capturando sua essência visual, UX e funcionalidade.

> **Escopo:** 18 × (JSON ultra-estruturado + HTML preview + Markdown doc + Code samples + Animações CSS + Auditoria WCAG AA + Benchmarking + 5 variações de cor + Guia responsivo) = **45.000+ linhas prontas para implementação imediata**

> **Qualidade:** Todos os DSs baseados em sites reais premiados em Awwwards, Webby Awards, CSS Design Awards, James Beard Foundation, NYT, INT Awards (2024-2026). Funcionais, testados, acessíveis.

---

## INSTRUÇÕES PARA A IA

Você vai fazer o seguinte:

1. **Pesquise na internet** os 3 sites mais premiados e famosos para CADA categoria abaixo
2. **Para cada site:** extraia cores exatas (HEX), tipografia (Google Fonts), layout, animações, voz
3. **Crie 1 Design System único por site** — NÃO sintetize, crie 3 totalmente distintos
4. **Gere os 9 entregáveis** por DS (JSON + preview + HTML + code + CSS + accessibility + benchmarking + colors + responsive)
5. **Valide** contra checklist de qualidade

### 🅰️ TEMA 01: ACOLHEDOR E FAMILIAR

**Segmento:** Padaria, escola, clínica de família, consultório pediátrico, cafeteria, floricultura, pet shop, consultório de veterinária

#### DS 01.1 — Padaria Artesanal Warm
- **Base:** Pinchmans Café (Sudbury, Canada) — Awwwards Nominee jun/2022
- **URL:** https://pinchmans.ca
- **Essência:** Terracota quente, serif elegante warm (Lora), fotos reais de pessoas, cards arredondados, espaço em branco, hero com fotografia
- **Cor primária HEX:** #B85C38 (terracota)
- **Cor secundária HEX:** #E8B696 (bege quente)
- **Tipografia heading:** Lora (serif elegant warm)
- **Tipografia body:** Inter
- **Animações:** Suaves, lentas (400-600ms), fade-in sutis
- **Público:** Famílias, pessoas que valorizam proximidade e calor humano
- **Por que:** Comunica "sou bem-vindo, este lugar me trata como pessoa" desde o primeiro segundo

#### DS 01.2 — Clínica Pediátrica Friendly
- **Base:** Pediatric clinic premium design ou site de pediatra high-end
- **Essência:** Azul-céu suave, ilustrações hand-drawn, depoimentos de pais, equipe visível, design convidativo
- **Cor primária HEX:** #87CEEB (azul-céu)
- **Cor secundária HEX:** #FFE4B5 (amarelo-manteiga)
- **Tipografia heading:** Nunito (arredondada, amigável)
- **Tipografia body:** Karla
- **Animações:** Brinquedonas mas suaves, não frenéticas
- **Público:** Pais, crianças, pessoas que buscam acolhimento médico

#### DS 01.3 — Floricultura Delicada
- **Base:** Floral boutique premium ou luxury flower shop
- **Essência:** Rosa antigo, verde-jade, fotografia natural delicada, grid de flores, hero fotografado, textura suave

### 🅳 TEMA 04: TÉCNICO E CONFIÁVEL

**Segmento:** SaaS, fintech, contabilidade, clínica médica, engenharia, consultoria B2B, agência de tecnologia, escritório de software

#### DS 04.1 — SaaS Minimalista
- **Base:** Linear (dark mode, tipografia ousada, animações funcionais)
- **URL:** https://linear.app
- **Essência:** Dark mode, monospace, grid quebrado assimétrico, animações funcionais (não decorativas), focus em produto
- **Cor primária HEX:** Roxo elétrico ou cor brand (#5E3DD2 exemplo)
- **Cor secundária HEX:** Off-white em dark (#FFFFFF)
- **Tipografia heading:** PP Neue Montreal (geometric bold)
- **Tipografia body:** Inter (clara, legível)
- **Animações:** Suaves, funcionais (300-400ms), não decorativas, ease-out
- **Layout:** Full viewport hero, cards sem borders, focus em produto screenshots
- **Público:** Developers, tech teams, CTOs, decisores B2B tech
- **Por que:** "Estes profissionais sabem o que estão fazendo" — confiança através de clareza, não marketing

#### DS 04.2 — Fintech Confiança
- **Base:** Stripe (espaço em branco, ilustrações geométricas, hierarchy clara)
- **URL:** https://stripe.com
- **Essência:** Light mode, Navy + Green estabilidade, ilustrações geométricas únicas, máximo espaço em branco, grids simétricos
- **Cor primária HEX:** #0F3460 (navy)
- **Cor secundária HEX:** #16A085 (verde-estabilidade)
- **Tipografia heading:** IBM Plex Sans (neutral, corporate clarity)
- **Tipografia body:** Söhne (refinada)
- **Imagens:** Dashboards, infográficos, screenshots de produto, diagramas
- **Animações:** Mínimas, fade-ins, reveal on scroll sutis
- **Público:** Empresas B2B, integrators, decision makers financeiros
- **Por que:** Confiabilidade comunicada através de clareza + dados + espaço

#### DS 04.3 — Contabilidade Premium
- **Base:** Crowe (contabilidade global) ou Pearl Accountants (design limpo + badges)
- **URL:** https://crowe.com ou https://pearlaccountants.com
- **Essência:** Charcoal + dourado-sutil, dados como protagonistas, badges de credibilidade, prova social com números
- **Cor primária HEX:** #2C3E50 (charcoal)
- **Cor secundária HEX:** #D4AF37 (dourado-sutil)
- **Tipografia heading:** GT America (neutral geometric)
- **Tipografia body:** IBM Plex (mono optional para dados)
- **Elementos:** Badges de credibilidade (QuickBooks Platinum, ICAEW), números reais, depoimentos
- **Animações:** Mínimas, counter animations, fade-ins
- **Público:** Pequenos negócios, empreendedores, CFOs
- **Por que:** Dados específicos > promessas vagas. Badges de credibilidade reduzem fricção.

## Overview
Design System para padarias, cafeterias artesanais e negócios acolhedores. Baseado em Pinchmans Café (Awwwards Nominee).

Terracota quente + serif elegante + fotos reais. O design desaparece; o afeto fica.

### Secundária: Bege Quente
- **Cor:** #E8B696
- **Uso:** Supporting elements, secondary buttons, accents
- **Contraste:** 4.8:1 (AA) ✅
