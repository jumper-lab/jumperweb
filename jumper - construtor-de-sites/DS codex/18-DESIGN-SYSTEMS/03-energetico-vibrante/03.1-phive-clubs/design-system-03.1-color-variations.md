# Color Variations — Phive Kinetic Membership System

As variações mantêm a assinatura kinetic type, class energy, membership scoreboard. Elas não autorizam transformar o DS em outro estilo.

## Dark mode

| Token | HEX |
|---|---|
| Primary | #A1BF00 |
| Secondary | #1D1D1D |
| Surface | #010101 |
| Text | #FFFFFF |

Preview: `#A1BF00` + `#1D1D1D` sobre `#010101`.

Uso recomendado: preservar scoreboard plus class grid e class-timetable-grid como âncoras visuais.
