# Benchmarking — Phive Kinetic Membership System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | scoreboard plus class grid cria uma experiência própria em vez de template genérico. |
| Navegação | training-mode nav with sticky join timer reduz fricção no caminho principal. |
| Hero | oversized condensed type over training motion, class chips and membership timer prova a proposta já na primeira dobra. |
| Componentes | class-timetable-grid, membership-scorecard, trainer-action-card, challenge-marquee, zone-map, join-timer-button dão vocabulário visual exclusivo. |
| Motion | fast diagonal wipe, class chip snap, hover punch reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de class-timetable-grid devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Membership scoreboard, Class timetable, Trainer cards, Facility zones, Challenge strip, Plan comparison, Join form, Footer as club board deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Academia / fitness club
- fitness buyers motivated by momentum

## Riscos Evitados

- avoid generic black-neon SaaS styling
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | scoreboard plus class grid | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | training-mode nav with sticky join timer | O menu precisa carregar uma função do segmento. |
| Hero | oversized condensed type over training motion, class chips and membership timer | A mídia/diagrama precisa provar a promessa. |
| Motion | fast diagonal wipe, class chip snap, hover punch | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | horizontal class scroller and sticky join button | Mobile preserva conversão e leitura. |
