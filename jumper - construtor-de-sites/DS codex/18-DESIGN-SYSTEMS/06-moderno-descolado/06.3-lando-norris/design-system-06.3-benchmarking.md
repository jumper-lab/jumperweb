# Benchmarking — Lando Velocity Fan System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | telemetry commerce grid cria uma experiência própria em vez de template genérico. |
| Navegação | race-week nav with merch/drop ticker reduz fricção no caminho principal. |
| Hero | speed-line editorial grid with race status, merch drop and fan CTA prova a proposta já na primeira dobra. |
| Componentes | race-status-strip, merch-drop-card, telemetry-stat-tile, fan-club-panel, calendar-race-row, paddock-pass-footer dão vocabulário visual exclusivo. |
| Motion | speed-line sweep, ticker scroll, card overtake hover reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de race-status-strip devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Race week status, Merch drop wall, Content feed, Fan club prompt, Calendar strip, Partner proof, Shop conversion, Footer as paddock pass deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Cultura jovem / esportes / merchandise
- motorsport fans and merch buyers

## Riscos Evitados

- avoid generic sports blog layout; commerce and race rhythm must lead
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | telemetry commerce grid | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | race-week nav with merch/drop ticker | O menu precisa carregar uma função do segmento. |
| Hero | speed-line editorial grid with race status, merch drop and fan CTA | A mídia/diagrama precisa provar a promessa. |
| Motion | speed-line sweep, ticker scroll, card overtake hover | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | race status stays sticky, merch drops become swipe lane | Mobile preserva conversão e leitura. |
