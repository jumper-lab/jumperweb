# Monolith Ink Portfolio System

Categoria: 06 — Moderno e Descolado
Site inspiração: Monolith Studio — https://monolithstudio.com/

Este DS não é uma pele de cores. Ele parte do padrão near-black portfolio wall, artist authorship, volumetric mark e transforma o site em um sistema com anatomia própria: portfolio monolith wall.

## Filosofia Visual

- Arquitetura: portfolio monolith wall
- Navegação: artist index rail with booking rules
- Hero: near-black immersive mark, massive type and rotating artist index
- Ritmo: slow 3D mark drift, portfolio mask reveal, magnetic artist rail
- Responsivo: portfolio wall becomes vertical artist archive
- Risco principal evitado: avoid nightclub neon that cheapens tattoo authorship
