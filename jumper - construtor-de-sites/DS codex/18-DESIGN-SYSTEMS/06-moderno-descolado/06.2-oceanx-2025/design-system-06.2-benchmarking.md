# Benchmarking — OceanX Expedition Interface System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | deep-scroll expedition interface cria uma experiência própria em vez de template genérico. |
| Navegação | mission timeline nav with depth markers reduz fricção no caminho principal. |
| Hero | full-bleed ocean scene with depth scale, mission chapter and interactive timeline prova a proposta já na primeira dobra. |
| Componentes | depth-marker-nav, mission-chapter-card, vessel-map-panel, discovery-log-entry, crew-signal-card, coordinate-footer dão vocabulário visual exclusivo. |
| Motion | depth parallax, sonar ping, chapter crossfade reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de depth-marker-nav devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Mission chapter, Depth timeline, Research vessel map, Crew cards, Discoveries log, Media theatre, Join/support prompt, Footer as expedition coordinates deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Experiência imersiva / produtora cultural
- culture, science and media audiences

## Riscos Evitados

- avoid generic space-tech aesthetics; keep ocean evidence visible
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | deep-scroll expedition interface | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | mission timeline nav with depth markers | O menu precisa carregar uma função do segmento. |
| Hero | full-bleed ocean scene with depth scale, mission chapter and interactive timeline | A mídia/diagrama precisa provar a promessa. |
| Motion | depth parallax, sonar ping, chapter crossfade | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | depth timeline becomes chapter stack with sticky coordinates | Mobile preserva conversão e leitura. |
