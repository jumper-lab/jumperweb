# Benchmarking — Zepto Payment Flow System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | transaction-flow map cria uma experiência própria em vez de template genérico. |
| Navegação | product nav with trust and developer branches reduz fricção no caminho principal. |
| Hero | transaction stream hero showing money movement, risk checks and API proof prova a proposta já na primeira dobra. |
| Componentes | transaction-flow-line, risk-control-card, api-snippet-panel, compliance-badge-row, use-case-route-card, demo-request-form dão vocabulário visual exclusivo. |
| Motion | line-trace payment path, calm data reveal, no flashy finance gimmick reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de transaction-flow-line devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Payment flow explainer, Trust and compliance panel, Developer integration, Use-case routes, Risk controls, Customer proof, Demo request, Footer as compliance ledger deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Fintech / pagamentos
- finance operators, product teams and compliance buyers

## Riscos Evitados

- avoid crypto/neon cues that reduce institutional trust
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | transaction-flow map | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | product nav with trust and developer branches | O menu precisa carregar uma função do segmento. |
| Hero | transaction stream hero showing money movement, risk checks and API proof | A mídia/diagrama precisa provar a promessa. |
| Motion | line-trace payment path, calm data reveal, no flashy finance gimmick | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | flow map becomes numbered transfer sequence | Mobile preserva conversão e leitura. |
