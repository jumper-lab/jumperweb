# Benchmarking — Mygom Product Cockpit System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | dashboard cockpit cria uma experiência própria em vez de template genérico. |
| Navegação | workspace nav with product tour anchor reduz fricção no caminho principal. |
| Hero | dashboard frame hero with task layers, proof pills and tour CTA prova a proposta já na primeira dobra. |
| Componentes | dashboard-frame, feature-stack-row, workflow-node-card, comparison-ledger, security-proof-pill, trial-start-form dão vocabulário visual exclusivo. |
| Motion | dashboard layer parallax, node connect, restrained hover glow reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de dashboard-frame devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Product cockpit, Feature stack, Workflow automation, Comparison table, Security proof, Customer scenarios, Trial form, Footer as workspace map deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- SaaS / produto digital
- operators comparing SaaS tools

## Riscos Evitados

- avoid empty gradient SaaS cards without workflow context
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | dashboard cockpit | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | workspace nav with product tour anchor | O menu precisa carregar uma função do segmento. |
| Hero | dashboard frame hero with task layers, proof pills and tour CTA | A mídia/diagrama precisa provar a promessa. |
| Motion | dashboard layer parallax, node connect, restrained hover glow | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | dashboard screenshot crops into annotated cards | Mobile preserva conversão e leitura. |
