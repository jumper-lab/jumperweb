# Benchmarking — Halo Precision Theatre System

## Por Que Este DS Converte Melhor

| Fator | Otimização |
|---|---|
| Anatomia | center-stage product theatre cria uma experiência própria em vez de template genérico. |
| Navegação | minimal command bar with consultation drawer reduz fricção no caminho principal. |
| Hero | black stage with centered treatment object, orange halo cue and consultation drawer prova a proposta já na primeira dobra. |
| Componentes | halo-focus-button, treatment-timeline, before-after-slider, doctor-authority-card, concierge-drawer, pricing-confidence-row dão vocabulário visual exclusivo. |
| Motion | spotlight aperture, precise mask reveal, no playful motion reforça o comportamento do segmento. |

## Performance

- LCP: menor que 2.5s; hero precisa usar imagem otimizada ou CSS leve.
- INP: menor que 200ms; interações de halo-focus-button devem ser simples.
- CLS: menor que 0.1; reservar dimensões de mídia e módulos.

## SEO

- Título com segmento + diferencial real.
- Schema.org adequado ao negócio final.
- FAQs somente quando houver perguntas úteis.
- Conteúdo de Signature treatment theatre, Before-after evidence, Doctor authority, Treatment timeline, Financing confidence, Consultation drawer, Press/proof strip, Footer as concierge deve virar heading hierarchy, não apenas blocos visuais.

## Casos Ideais

- Clínica dental premium
- high-intent patients evaluating premium treatment

## Riscos Evitados

- avoid overdecorated luxury gold and generic dental smiles
- Repetir grid, hero ou card de outro DS.
- Usar referência como molde literal.


## Comparação Contra Template Genérico

| Camada | Decisão específica deste DS | Como validar |
|---|---|---|
| Layout | center-stage product theatre | A primeira dobra não pode encaixar em outro DS sem perder sentido. |
| Navegação | minimal command bar with consultation drawer | O menu precisa carregar uma função do segmento. |
| Hero | black stage with centered treatment object, orange halo cue and consultation drawer | A mídia/diagrama precisa provar a promessa. |
| Motion | spotlight aperture, precise mask reveal, no playful motion | Movimento deve reforçar comportamento, não decorar. |
| Responsivo | dark single stage with drawer CTA below fold | Mobile preserva conversão e leitura. |
