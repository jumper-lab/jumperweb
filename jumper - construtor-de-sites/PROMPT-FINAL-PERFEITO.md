# 🎯 PROMPT DEFINITIVO: CRIAR 18 DESIGN SYSTEMS PREMIUM

> **Versão:** 2.0 Final - Qualidade Total  
> **Para:** Qualquer IA com capacidade de web search  
> **Objetivo:** Pesquisar na web, analisar e gerar 18 Design Systems completamente distintos — 3 para cada uma das 6 personalidades visuais. Cada DS baseado em um site premiado/famoso diferente do mundo real.

> **Output:** 18 × (JSON + HTML preview + Markdown + Code samples + Animações + Accessibility audit + Benchmarking + Color variations + Responsive guide) = **162 arquivos, 45.000+ linhas prontas para implementação**

---

## 🔍 FASE 1: PESQUISA WEB (VOCÊ FARÁ ISTO)

Para cada uma das 6 personalidades abaixo, você vai:

1. **Pesquisar na internet** os 3 sites MAIS premiados e funcionais daquela categoria
2. **Critérios de seleção:**
   - ✅ Premiados em Awwwards, Webby Awards, CSS Design Awards, INT Awards, James Beard Foundation, NYT (2024-2026)
   - ✅ Funcionais (não apenas bonitos — conversão, UX, performance comprovada)
   - ✅ Publicamente acessíveis (sem paywalls)
   - ✅ Representam melhor-em-classe (não genéricos)
   - ✅ Visualmente distintos entre si (mesmo segmento, execuções diferentes)

3. **Para cada site, documente:**
   - URL exata
   - Prêmios e reconhecimento (Awwwards, Webby, etc)
   - Design studio/autor
   - Padrão visual identificado
   - Por que é referência

4. **Acesse cada site e extraia:**
   - **Cores:** Use ferramentas (WebAIM Color Contrast Checker, Stark, ou screenshot + análise) para extrair HEX exatos
   - **Tipografia:** Identifique fonts usando inspector (Google Fonts preferred)
   - **Layout:** Descreva grid, composição, proporções
   - **Componentes:** Documente botões, cards, inputs, heróis, footers
   - **Animações:** Capture padrões (fade-in duration, hover behavior, easing)
   - **Voice/Tone:** Analise copy — headlines, CTAs, tom de linguagem
   - **Público-alvo:** Quem este site atrai?
   - **Riscos evitados:** O que este site NÃO faz (contraste fraco, slow animations, etc)
